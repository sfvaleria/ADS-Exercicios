# Repositório de Projetos Acadêmicos – ADS

Este repositório tem como objetivo organizar e documentar os projetos desenvolvidos ao longo da graduação em **Análise e Desenvolvimento de Sistemas (ADS)**.

Os projetos estão estruturados em **pastas por semestre**, permitindo acompanhar a evolução do aprendizado em programação, análise de dados e desenvolvimento de soluções computacionais.

---

## Estrutura do Repositório

Cada diretório representa um semestre do curso e contém os projetos, exercícios e atividades práticas desenvolvidas nas disciplinas correspondentes.
